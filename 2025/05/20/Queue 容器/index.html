Queue 容器
发表于 2025-05-20 | 更新于 2025-06-03
重写STL
STL | Container | C++

​ STL中的Queue容器是基于别的容器封装的，默认实现是基于deque封装实现了，但基于苯人的学习于实现顺序，我们基于List容器封装实现一个Queue容器。

实现如下：

成员函数 功能描述
void push(const T&) 向队尾添加一个元素（入队）
void pop() 移除队首元素（出队）
T& front() 返回队首元素的引用，队列为空时行为未定义
const T& front() const 返回队首元素的常量引用
T& back() 返回队尾元素的引用，队列为空时行为未定义
const T& back() const 返回队尾元素的常量引用
bool empty() const 判断队列是否为空
std::size_t size() const 返回队列中的元素数量
