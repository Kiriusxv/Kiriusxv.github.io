List 容器
发表于 2025-05-16 | 更新于 2025-06-03
重写STL
STL | Container | C++

​ List容器其实就是一个双向链表：一个由节点组成的线性的数据结构，每个节点都存储一个元素，并且包含指向前一个和后一个元素的指针。

​ 这种独特的结构使得它区别于vector拥有一些独特的优缺点：插入/删除效率高——任意位置都为O(1)，但是不支持随机访问——无法通过下标访问元素，不需要连续的存储——不需要移动元素，也不涉及扩容复制。

​ 常规操作如push_back,push_front，其原理就是创建一个新节点挂载到尾部的next或者头部的next，pop类似。

​ 而insert/erase则需要通过迭代器获取目标位置，再修改相邻的节点的指针来实现插入和删除，这种方式支持任意位置

实现的功能如下：

成员函数 功能描述
list() 默认构造函数，初始化一个空链表
~list() 析构函数，释放链表中所有节点
push_back() 在链表尾部插入一个元素
push_front() 在链表头部插入一个元素
pop_back() 移除链表尾部元素
pop_front() 移除链表头部元素
front() 返回头部元素的引用，若链表为空则抛异常
back() 返回尾部元素的引用，若链表为空则抛异常
empty() 判断链表是否为空（即 size == 0）
size() 返回当前链表中的元素个数
clear() 清空整个链表，释放所有节点并重置状态
